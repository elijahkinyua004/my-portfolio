# ELIJAH GRAPHICS Portfolio

This is the portfolio website for ELIJAH GRAPHICS, a professional graphic designer and web developer based in Ruiru. The website showcases the latest projects, skills, and services offered by ELIJAH GRAPHICS.

## Features

- Responsive design for mobile and desktop
- Portfolio gallery with project previews
- Contact form for inquiries and project requests
- Pricing information for design and printing services

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Twilio


## Installation

1. Clone the repository: `git clone https://github.com/your-username/portfolio.git`
2. Install dependencies: `npm install`
3. Start the server: `npm start`

## Usage

- Visit `http://localhost:3000` in your browser to view the website
- Click on project cards to view project details and contact information
- Use the contact form to send inquiries or project requests
